# Compliance and Security Touchpoints (UX Surface)

Design requires explicit UI surfaces for:

- Data privacy. Transparent consent screens and clear access to privacy policy.
- Data sovereignty. User control and clarity over what is shared.
- Certifications and legal. Where required, ensure discoverable "About" or legal section includes labels and disclosures.
