# JTBD and Success Metrics

## Core Jobs

- Book appointment near me quickly without phone calls.
- Consult a doctor via video for non-physical concerns.
- Redeem an e-prescription with minimal effort. Choose online delivery or local pharmacy.

## Supporting Jobs

- Track appointments and prescriptions in one place.
- Understand costs before committing.
- Discreet handling for sensitive needs.

## Success Metrics (Product)

- Booking. Completed booking, then showed up.
- Prescription online. Delivered successfully, delivery time.
- Prescription offline. Redeemed successfully, time to find pharmacy.
- Telemedicine. Completed call, user satisfaction.
