## Phase 6: Pharmacy Search - Offline (3 prompts)

### Prompt 6A: Enable Offline Option

```
Update MedAlpha prototype:

On Screen RX-01 (Redemption Type):
- Card 2 - Local Pharmacy should navigate to PS-01

Add Screen PS-01 - Pharmacy Map:
- Header with back arrow: "Find a Pharmacy"
- Map area (large, ~60% of screen):

```
